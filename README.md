# Enterprise Agentic AI Platform

Interactive black-and-white reference architecture diagram for an enterprise agentic AI platform.

Click any block to open a side panel with responsibilities, spec, and cross-cutting touchpoints. Hover the cross-cutting chips (Governance / Observability / Identity) to halo every block they govern. Press `Esc` to deselect.

Single static page — open `index.html` in any browser, or view the deployed build.
